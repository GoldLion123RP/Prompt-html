This is an extremely common and very weak exam prep prompt. It will almost always return a generic, unprioritised list of topics, with no distinction between "this will be 30% of the exam" and "this will never be asked".

Below is the enhanced, battle tested version that will consistently give you dramatically better, more actionable output. I have also included specialised variants for different exam types and situations.

Flagship Enhanced Prompt

This is the default version you should use 90% of the time:

I have the full content for these 2 modules below. I am studying for a closed book university exam.

Act as a senior lecturer and former exam writer for this subject. Tell me exactly what I need to learn, prioritised strictly by actual exam yield.

Rules:

Key improvements made to the original prompt:

The original prompt fails because it never tells the LLM what good output looks like. These changes are responsible for almost all of the improvement:

  1. Gives the model a specific, extremely useful persona: former exam writer, not generic tutor
  2. Enforces an explicit prioritisation framework, most importantly the almost never requested tier for content you can ignore
  3. Explicitly bans all the useless filler output LLMs love to give
  4. Asks for the single most valuable exam information that no one ever asks for: common traps and trick questions
  5. Gives an exact output structure so you get a scannable, usable answer not a wall of text
