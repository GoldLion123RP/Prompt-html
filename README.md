## Key Design Decisions

1. **Specific persona** ‚Äî Former exam writer, not generic tutor
2. **Explicit prioritisation framework** ‚Äî Including what to skip entirely
3. **Filler banned** ‚Äî Removes useless "you should also be familiar with..." padding
4. **Trap questions** ‚Äî The most valuable info no one asks for
5. **Exact output structure** ‚Äî Scannable answer, not a wall of text
